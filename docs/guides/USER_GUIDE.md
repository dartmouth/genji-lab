## User Roles

### Student Role

**What students can do:**
- View assigned document collections
- Read documents
- Create annotations
- Respond to classmates' annotations
- Search documents and annotations
- Join classrooms

**What students cannot do:**
- Create or edit document collections
- Upload documents
- Manage classroom membership
- Access admin features

**Learn more:** [Student Guide](STUDENT_GUIDE.md)

### Instructor Role

**What instructors can do:**
- All student capabilities, plus:
- Create and manage classrooms
- Upload and organize documents
- Assign documents to classrooms
- Monitor student annotations
- Manage classroom membership

**Learn more:** [Instructor Guide](INSTRUCTOR_GUIDE.md)

### Administrator Role

**What administrators can do:**
- All instructor capabilities, plus:
- Manage all users and roles
- Configure site settings
- Review flagged content
- Access system-wide reports
- Manage all classrooms and documents

**Learn more:** [Administrator Guide](ADMIN_GUIDE.md)

## Common Tasks

### Reading a Document

[Step-by-step instructions]

1. Navigate to Document Collections
2. Select a collection
3. Click on a document to open
4. Use scroll or navigation to read
5. [Any special features like zoom, print, etc.]

### Creating an Annotation

[Step-by-step instructions]

1. Open a document
2. Select the text you want to annotate
3. Right-click or use the annotation menu
4. Choose annotation type (comment, tag, link, etc.)
5. Enter your annotation content
6. Click "Save" or "Submit"

**For detailed annotation instructions, see:** [Annotations Guide](ANNOTATIONS_GUIDE.md)

### Searching for Content

[Step-by-step instructions]

1. Click the Search icon or press Ctrl+F (Cmd+F on Mac)
2. Enter your search query
3. Use filters to narrow results:
   - By collection
   - By document type
   - By annotation type
4. Click on a result to view it in context

### Responding to Annotations

[Step-by-step instructions]

1. Navigate to the document with the annotation
2. Find the annotation in the annotations panel
3. Click "Reply" or the reply icon
4. Enter your response
5. Click "Submit"

### Flagging Inappropriate Content

[Step-by-step instructions]

1. Find the annotation or content to flag
2. Click the flag icon or "Report" button
3. Select a reason for flagging
4. Add any additional context (optional)
5. Submit the flag

**Note:** Flagged content is reviewed by instructors/administrators.
